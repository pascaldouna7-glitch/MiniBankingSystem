# 🏦 Mini Banking System (C++)

## 📌 Overview

A fully functional **console-based banking system** built using C++ and Object-Oriented Programming.
This system simulates real banking operations with persistent storage and security.

---

## 🚀 Features

* Create secure bank accounts (with PIN)
* Deposit and withdraw money
* Balance tracking
* Transaction history
* Data persistence using file handling
* Colored console interface

---

## 🔐 Security

Each account is protected with a **PIN authentication system**, ensuring secure transactions.

---

## 💾 Data Storage

All account data is saved in:

```id="f5vt05"
accounts.txt
```

Includes:

* Account Number
* Name
* PIN
* Balance

---

## 🧱 Technologies Used

* C++
* OOP (Encapsulation, Classes, Objects)
* File Handling
* STL (Vectors)
